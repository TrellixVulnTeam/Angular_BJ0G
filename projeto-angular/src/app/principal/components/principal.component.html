O que é o Angular ?

 

Angular é um poderoso framework para construção que utiliza HTML, CSS, Javasript tipografada em TypeScript para criar a interface com o usuário, ou seja, o front-end em aplicações web, desktop e dispositivos móveis, utilizado sempre como uma SPA (single page application) em suas criações.

A primeira versão foi lançada em 2010 com o nome Angular JS, mas foi completamente reescrita e em 2016 passou a ser chamada Angular 2. Atualmente já encontramos em utilização a versão Angular 12
